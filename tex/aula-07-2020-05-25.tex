\section{Gnuplot (continuação)}\label{sec:gnuplot}

O que nós já vimos até agora sobre Gnuplot?
\begin{itemize}
\item Gnuplot é uma pequena linguagem gráfica de programação.
\item É preciso instalar o programa, e ele precisa estar no seu caminho
de programs executáveis (PATH).
\item Gnuplot interage com \LaTeX, e possibilita o uso de símbolos matemáticos
nos eixos, em legendas, etc..
\item Eu fiz um pequeno arquivo de \textit{batch}, denominado \texttt{epslatex.bat} em Windows,
para automatizar a geração de figuras em pdf.
\item \texttt{epslatex.bat} exige que nós façamos um \textit{script *.plt} por figura.
\end{itemize}

%
% isso é um comentário
% 

Eis aqui o arquivo epslatex.bat:
\begin{verbatim}
rem -------------------------------------------------------
rem epslatex compiles gnuplot/epslatex scripts 
rem that use the new standalone option
rem -------------------------------------------------------
gnuplot %1.plt
latex %1         rem run latex (make sure eps is generated)
dvips %1
eps2eps %1.ps  %1.eps
epstopdf %1-inc.eps
pdflatex %1      rem run pdflatex 
rem --------------------------------------------------------
rem check this out *very* carefully
rem --------------------------------------------------------
del %1.dvi
del %1-inc.eps
del %1-inc.pdf
del %1.aux
del %1.log
del %1.tex
\end{verbatim}
